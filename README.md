# 🐍 Snake Rush - Classic Arcade Game

A modern take on the classic **Snake Game**, built using HTML, CSS, and JavaScript. Control the snake, eat food, grow longer, and try to achieve the highest score without colliding!

---

## 🚀 Live Demo

 - Unmute and play the video to enjoy the game.



https://github.com/user-attachments/assets/249ac481-5cf6-4561-abc6-27b3fd8bf976



## 🎮 Gameplay

* 🐍 Control the snake
* 🍎 Eat food to grow
* 📈 Score increases with each food item
* 💥 Game ends on collision with wall or itself

---

## ✨ Features

* 🎯 Smooth snake movement
* 🍎 Random food generation
* 📊 Live score tracking
* ⚡ Increasing difficulty 
* 🎮 Responsive controls
* 🎵 Sound effects
* 🏆 High score system

---

## 🕹 Controls

* ⬆️ Up
* ⬇️ Down
* ⬅️ Left
* ➡️ Right

---

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript

---

## ⚙️ How It Works

* JavaScript updates snake position at regular intervals
* Food appears at random positions
* Collision detection checks:

  * Wall collision
  * Self collision
* Score updates dynamically
